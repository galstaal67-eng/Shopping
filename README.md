# 🛒 אפליקציית רשימת קניות

אפליקציית ווב (קובץ HTML יחיד, ללא שרת) שנבנתה לפי העיקרון של הקובץ
"רשימת קניות.xlsx" — קטלוג מוצרים קבוע לפי קטגוריות, סימון מה שצריך,
ורשימת קנייה נוחה למי שמבצע את הקנייה.

## שימוש

פותחים את `index.html` בדפדפן (בנייד או במחשב). הנתונים נשמרים
בדפדפן (localStorage) — אין צורך בהתקנה או חשבון.

### מסך "מה צריך" 📝
- קטלוג המוצרים מהאקסל המקורי (כולל ההערות), מועשר במוצרים נפוצים נוספים
  ובקטגוריות חדשות (לחם ומאפים, קפואים ושתייה). קטגוריית הירקות מפוצלת
  לשלוש: ירקות 🍅, שורשים ובצלים 🥔, עלים ותבלינים 🌿.
- מסמנים ✓ מוצר כדי להוסיף לרשימה, ואפשר להוסיף לו הערה (כמות, סוג...).
- **בקשות מיוחדות ⭐** — קטגוריה חופשית לבקשות חד-פעמיות.
- הוספת מוצרים חדשים לקטלוג בכל קטגוריה; "עריכת קטלוג" מאפשרת מחיקה.
- "שלח לקונה" — שולח את הרשימה המסודרת בוואטסאפ.
- "איפוס רשימה" — מנקה סימונים ובקשות מיוחדות לקראת הסבב הבא (הקטלוג נשמר).

### מסך "קנייה" 🛍️
- רק הפריטים שסומנו, מקובצים לפי קטגוריות מתקפלות, עם מד התקדמות.
- הקשה על פריט מסמנת "נאסף" (הפריטים שנאספו יורדים לתחתית הקטגוריה).
- **מעקב מחיר ₪** — סימון פריט כ"נאסף" פותח אוטומטית את חלון המחיר:
  צילום תווית המחיר שעל המדף או השקית (זיהוי ספרות אוטומטי, עם אפשרות
  תיקון) או הזנה ידנית. הסכום המצטבר של העגלה מוצג למעלה ונכלל
  בהודעת "סיימתי".
- **⏱ מדידת זמן** — "התחל קנייה" מפעיל שעון עצר; "🏁 סיים" שומר את
  הקנייה בהיסטוריה.
- **🏷️ מבצעים** — כפתור שמציג מבצעים מהחנות (כרגע הדגמה; מתוכנן חיבור
  לחנויות שיזהו את הקונה וישדרו מבצעים בזמן אמת).
- כפתור וואטסאפ ליד כל פריט — שולח שאלה למי שרשם את הרשימה, עם תבניות
  מוכנות (למשל: "אין עגבניות שרי — להביא משהו דומה במקום?") או ניסוח חופשי.
- בסיום — עדכון "סיימתי" בוואטסאפ ואיפוס לרשימה הבאה.

### מסך "היסטוריה" 🧾
- כל קנייה שהסתיימה נשמרת: תאריך ושעה, המוצרים, המחירים, הסכום ומשך
  הקנייה.
- **📥 ייבוא לרשימה חדשה** — מתחילים קנייה חדשה מרשימה קודמת בלחיצה.

### הגדרות ⚙️
- שם ומספר וואטסאפ של רושם/ת הרשימה — כפתורי השאלה יפתחו ישר את הצ'אט הנכון.
- אתחול מלא — החזרת קטלוג ברירת המחדל.

## הערות טכניות
- הכול בקובץ אחד: `index.html` (HTML + CSS + JS). זיהוי המחיר מהתמונה
  משתמש ב-tesseract.js שנטען מ-CDN רק בשימוש הראשון; ללא אינטרנט —
  הזנה ידנית עדיין עובדת.
- מפתח האחסון: `shopping-app-v1` ב-localStorage (סכמה v2; נתוני v1
  עוברים מיגרציה אוטומטית).
- שליחת וואטסאפ דרך קישורי `wa.me`; מספרים ישראליים (05X...) מנורמלים
  אוטומטית לפורמט בינלאומי (972...).
